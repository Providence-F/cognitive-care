# 认知维护系统 · 交互原型

> 为高强度学习者设计的"身体与自然连接仪式"。
> 非传统运动App，是学习后的废墟时间里的零决策成本轻运动协议。

---

## 🎨 设计风格说明

本原型融合了 GitHub 上高评分设计体系的精华：

| 设计来源 | 应用方式 |
|---------|---------|
| **mcp-vibe-ui: Japanese Minimalist** | 禅意骨架、最小决策成本、自然色调 |
| **mcp-vibe-ui: Organic Flow** | 流体曲线、有机形状、风/水/光动效 |
| **zen.nvim 配色盘** (1.2k⭐) | sage绿/sand米/sky蓝/lavender紫自然六色 |
| **simple_morph Glassmorphism** (800+⭐) | 毛玻璃质感、呼吸感层级、沉浸氛围 |

整体风格：**清新、轻盈、有诗意**，零AI味，手工编排动画曲线。

---

## 📱 如何运行

**零依赖、零构建、双击即运行**

1. 直接用浏览器打开 `index.html` 文件
2. 推荐使用 Chrome / Edge / Safari 最新版
3. 桌面端会自动居中显示手机外框
4. 移动端可全屏体验

---

## 🗺️ 页面导航

原型共 **4 个页面**，点击对应元素即可切换：

### 页面1 · 首页
- 顶部问候语（根据真实时间自动切换时段）
- 左侧 6 维雷达图（柔韧/力量/平衡/呼吸/专注/恢复）
- 右侧今日推荐协议卡片（玻璃拟态 + SVG自然插画）
- 底部时间导航（4时段切换，可手动点击）
- 中央大按钮「开始今日仪式」
- 队友动态（模拟环境共在）

### 页面2 · 协议执行页
- 顶部共鸣标签「此刻N人正在做...」
- 全屏自然意象 SVG 插画
- 横向滑动步骤卡片（动作 + 呼吸提示）
- 倒计时圆环 + 进度动画
- 「跳过此步」（始终可见，零惩罚）/「完成」
- 完成后粒子填充动画 + 维度增长提示

### 页面3 · 协议库页
- 4 个时间节律分区（手风琴展开）
- 协议卡片网格（已解锁/未解锁状态）
- 未解锁显示条件提示

### 页面4 · 档案页
- 大型 6 维雷达图
- 徽章墙（横向滑动）
- 已解锁协议缩略图网格
- 影子队友列表（无时间/时长/能量值）
- 收到的知识卡片
- 设置开关

## 🎯 核心机制体验

| 机制 | 体验方式 |
|------|---------|
| **协议解锁** | 初始解锁每个时段第1个协议，完成3次同类型解锁第2个 |
| **6维雷达图** | 完成协议后对应维度 +1%~3%，只增不减 |
| **影子队友** | 首页和执行页显示「此刻N人正在做...」 |
| **知识卡片** | 完成协议后弹出清新短句卡片 |
| **零决策成本** | 首页只有一个大按钮，「跳过」不惩罚 |

## 💾 数据持久化

所有进度自动保存到浏览器 `localStorage`，刷新页面不丢失：
- 雷达图数值
- 协议完成次数
- 解锁状态
- 设置偏好

## 📝 设计哲学

> **用户不是不想动，是不想觉得自己是个废物。**

- 竞品不是 Keep，是抖音
- 命名即治疗：清新文案降低心理阻抗
- 跳过按钮不是功能缺陷，是设计美德
- 最高级的社交，是社交压力为零的社交
- 不完美的记录，比完美的记录更能驱动长期行为
